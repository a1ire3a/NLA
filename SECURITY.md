# Security Policy

## Supported Versions

This repository is a research prototype for a PhD recruitment task. It is not a production software package.

## Reporting a Vulnerability

If you discover a security issue, please do not open a public GitHub issue.

Instead, contact:

alireza.javaheri1376@gmail.com

I will review the report and respond as soon as possible.

## Data and Secrets

This repository intentionally does not include raw datasets, model checkpoints, activation tensors, or generated artifacts that may contain sensitive or third-party content.

Large local artifacts are excluded from version control through `.gitignore`.
